Começando estudos de CSS

Entendendo o que são Seletores e seus componentes

Seletores são os elementos presentes no HTML que vamos estilizar no CSS. Por exemplo:
h1 (é um seletor) {
background-color (uma propriedade): green (valor);
 (e todo esse conjunto é uma declaração) }

h2 pode ser um seletor também, div's, parágrafos, todos os títulos, se puder ter estilo e ser estilizado pode ser um seletor
Agora se tivermos mais de um h1 na nossa página e quisermos mexer apenas nele, podemos trabalhar com os "id's" ou com as classes.
Não podemos ter vários "id's" em um mesmo código HTML, segundo a WC3 está errado trabalhar dessa forma. Agora se quisermos que um mesmo elemento tenha as características do outro podemos utilizar as "classes" (representadas por um ponto ( . )).
id = único (por página HTML)
classe = múltiplo
Dentro de um seletor podemos ter id (sozinho), id e classe, class (sozinho) ou não ter nada (se aplica as configurações padrão).

Como diferenciar um ID de uma classe

Quando vamos trabalhar com ID's e classes em CSS temos uma diferenciação de como chama-las. Para ID's utilzamos "#" e para as classes utilizamos o ponto final ( . ).
Exemplo: < h1 id = " título-pincipal ">
Para chamar esse h1 em específico iremos utilzar a "#" no CSS
#título-pincipal {} (Já no CSS esse comando). Não é necessário colocar o nome da tag h1 antes da "#"
Agora para classes supondo que nosso site tenha dois h2, cada um com uma classe diferente.
< h2 class = " título-secundário " >
< h2 class = " Outro-título-secundário ">
As classes em CSS serão chamadas pelo "ponto final". Então em CSS ficará dessa forma:
. título-secundário {}
. Outro-título-secundário {}

Utilizando Hover

"Hover" é uma pseudoClasse que consiste no seguinte : sempre que passar o mouse pelo seletor que estiver com o "hover" ele realizará uma ação que será programada no momento que começarmos a estilziar o CSS.
Exemplo :
div:hover { color: darkblue; } -> aqui quando passarmos o mouse pelo elemento que estiver na div ele mudará de cor para "darkblue" .

Escondendo textos e utilizando hover

Aqui é uma forma que podemos começar a utilziar o "hover" para dar um certo dinamismo para o seu site. Quando criarmos uma "div" e colocarmos um parágrafo dentro dela podemos esconder esse parágrafo digitido.
1° Passo
< div >
< p > Texto Escondido < /p >
< /div >
Esse texto pode ser escondido utilizando CSS:
2° Passo :
div > p { display:none }: Isso significa que temos um parágrafo dentro de uma div (um parágrafo filho dessa div) .
Representado pelo sinal de maior ( > ). Aqui ele ainda não mostrará o texto escondido
Para que isso aconteça iremos utilizar o Hover.
3 °Passo :
div:hover > p {display : block} : utilizando o hover podemos mostrar o texto assim que passar o mouse por cima.

Pseudo-classe e Pseudo-elemento

Pseudo-classes : são palavras que definem estados de elementos dentro do HTML. São representadas nas CSS's com o "dois pontos ( : )".
Pseudo-elemento : eles podem agir nos elementos, nos id's, nas classes, eles mexem no conteudo periférico do elemento. São representados nas CSS's com o "2x dois pontos ( : : )"
