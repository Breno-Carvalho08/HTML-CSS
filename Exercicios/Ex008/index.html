Itálito/Ênfase

Seguindo a forma não semântica temos: Frase em Itálico

Utilizando a tag < i >.

Agora de forma semântica, prezando pelo significado, temos: Frase em Itálico.

Utilizando a tag < em > que significa "emphase".

Texto Marcado

Podemos criar também textos marcados

Fazemos isso utilizando a tag < mark >

Podemos alterar a cor da marcação utilizando o parâmetro dentro da tag mark

< mark style ="background-color: (cor desejada)" >. Essa é a forma em HTML5

Porem há uma forma de fazer em CSS, abrindo um seletor (mark), criando a propriedade (background-color)
e atribuindo o valor (limegreen).

Texto Grande e Pequeno

Criando textos grandes (não semântico) e textos pequenos (semântico)

Utilizando a tag < big >. Essa é uma tag obsoleta e não semântica.

Utilzando a tag < small >. Essa é uma tag semântica e pode ser utilizada.

Texto deletado e Inserido

Podemos indicar um texto como excluído para indicar que ele deve ser lido mas não deve ser considerado

Para texto excluido utilizamos a tag < del >

Podemos indicar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois

Para indicar um texto inserido ou sublinhado utilizamos a tag < ins >
