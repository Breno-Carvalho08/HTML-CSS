Anatomia do tipo

Quando vamos fazer um site é importante saber qual fonte utilizar para chamar mais atenção do usuário.

Temos atributos nas letras importantes para levar em conta, como: altura-x (letras minúsculas), altura das maiúsculas, ascendente, descencente e corpo

Anatomia Geometrica: haste, arco, filete, esporão, vértice, terminal.

Anatomia Humanizada: braço, perna, pé, espinha, bojo, oco, orelha, cauda.

Categorias de Fontes: Serifadas, sans-serif (sem serifas), monoespaçadas, handwiriting e display

Anatomia das fontes (passe o mouse)

Carregando Fontes externas

Caso não tenha uma fonte certa no VisualG que o seu cliente queira, podemos tanto utilizar uma fonte que está na internet como uma fonte que ele desenvolveu e disponibilizou para nós utilizarmos. Podemos fazer uso então do "@import" ou "@font-face".

@import: iremos utilizar o Google Fonts que nos dará uma grande variedade de fontes para utilizarmos.

@font-face: também podemos utilizar o Google Fonts, mas aqui o arquivo terá que ser baixado, e ele tera a seguinte estrutura.

@font-face {
font-family: (nome que nós iremos dar para colocar a fonte no nosso site)
src: url("Diretório do arquivo baixado") EX: src: url(Fontes/Arial.ttf)
}

Na parte do url ainda vamos colocar o "format()", que no caso do exemplo será "format('truetype')". Podemos ter ainda o ".otf" (opentype) que se fosse o caso substítuiria o ".ttf" e o "truetype".

Obs: sempre que vamos decidir a fonte que iremos usar é melhor colocarmos mais de uma, pois assim o navegador pode utilziar aquela que ele encontrar já que alguns podem não ter a fonte que você deseja. (font-family:Cambria, Cochin, Georgia, Times, 'Times New Roman', serif). Por isso é colocado dessa forma, como se fosse pulando de uma fonte para a aoutra até encontrar aquela que o site possuir.

Medidas das Fontes

Medidas absolutas: cm, mm, in (polegada), px (pixel), pt (ponto) (Utilizada no word), pc (paica). (pt e pc não é recomendado utilizar). Segundo a W3C devemos utilzar "px" ou "em".
1em = 16px

Medidas relativas: em (relativa a altura "m" de uma fonte), ex (relativa a altura "x" de uma fonte), rem, vw (view width), vh (view height), %

ShotHand:
font-style -> font-weight -> font-size -> font-family
font-style: italic;
font-weight: normal;
font-size: 1.5em;
font-family:Cambria, Cochin, Georgia, Times, 'Times New Roman', serif;
EX -> font: italic (font-style) bolder (font-weight) 2.5em (font-size) arial (font-family);
